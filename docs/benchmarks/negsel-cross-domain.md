## NegSel Performance Summary

NegSel (Negative Selection AIS, Umair et al. 2025) uses 10-dimensional dendritic
features extracted from raw text: claim_density, citation_count, hedging_ratio,
specificity, numeric_density, pamp_score, exaggeration, certainty, controversy,
and danger_signal.

These features are domain-agnostic (rule-based, no model dependency) but were originally
designed for hallucination-style anomaly detection. Performance on non-hallucination
domains reflects how well these dendritic signals discriminate anomalies in each domain.

**Key observations:**

- NegSel achieves perfect precision (1.000) on reasoning and bio domains, indicating zero false positives
- Low recall across all domains suggests the adaptive r_self threshold is conservative
- Fastest detector by latency (0.1-0.2ms per sample) due to simple distance-based classification
- Performance varies significantly by domain, confirming that dendritic features have domain-dependent discriminative power
- Best suited as a high-precision component in ensemble combinations rather than standalone
