## Ecosystem Observations

- **Barrier to entry:** Financially hard to start a subnet, which slows adoption of the Bittensor ecosystem
- **Bitstarter's advice:** Start with validation — define what you will validate before anything else
- This aligns with Antigence's approach: we validate AI output quality (the "what")
